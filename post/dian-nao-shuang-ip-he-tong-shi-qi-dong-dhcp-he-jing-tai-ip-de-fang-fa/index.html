电脑双IP和同时启动DHCP和静态IP的方法
2020-12-25

使用管理员模式打开cmd

启动DHCP和静态共存：
netsh int ipv4 set interface "以太网" dhcpstaticipcoexistence=enabled
配置IP地址：
netsh int ipv4 add address "以太网" 192.168.1.30 255.255.255.0 192.168.137.11

上面以太网为你的网卡名字 ，后面接着就是要设置的IP、子网掩码、网关
不设置网关可用其它网络上网，但也可以连你所设置的ip的局域网
